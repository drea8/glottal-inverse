## How to use

```sh
make BUILD_TYPE=Release
```
Builds release variant binaries.

```sh
make
# or
make BUILD_TYPE=Debug
```
Builds ddebug variant binaries.

```sh
make tests
```
Builds unit test binairies. Also supports `BUILD_TYPE`.

See `Makefile` for more make targets.
